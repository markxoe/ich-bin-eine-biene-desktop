## Selber Builden

### Auf Windows

Voraussetzungen:

- Computer
  - Prozessor
  - Bildschirm
  - Tastatur
  - Windows 10
- node & npm

1. Zuerst dieses Repository clonen: `git clone https://github.com/markxoe/ich-bin-eine-biene-desktop.git`
2. In den Ordner gehen: `cd ich-bin-eine-biene-desktop`
3. Benötigte Packages installieren: `npm i`
4. Den Build ausführen: `npm run build-win`
5. Sich freuen und das Setup (endung .exe) in `dist` ausführen
